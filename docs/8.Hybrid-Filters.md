# 🔥 Filtros Híbridos en EntityTable

Los **Filtros Híbridos** son una nueva funcionalidad que permite combinar filtros de checkbox (estilo Excel) con filtros tradicionales en una misma tabla, optimizando la experiencia de usuario según el tipo de datos.

## 🎯 ¿Qué son los Filtros Híbridos?

Los Filtros Híbridos permiten configurar **por columna** qué tipo de filtro usar:

- **📋 CheckBox Filters**: Lista de valores únicos con checkboxes (ideal para enums, booleans, categorías)
- **🔍 Traditional Filters**: Inputs con operadores (contains, equals, greater than, etc.) para texto libre y números

### ✅ Ventajas:

- **UX Optimizada**: Cada tipo de dato usa el filtro más apropiado
- **Performance**: Cache inteligente y lazy loading  
- **Excel-like**: Experiencia familiar para usuarios
- **Flexible**: Configuración granular por columna
- **Compatible**: No rompe funcionalidad existente

## ⚙️ Configuración Básica

### 1. Habilitar Filtros Híbridos

```xml
<EntityTable T="MiEntidad"
             EnableHybridFilters="true"
             DefaultFilterMode="FilterMode.CheckBoxList"
             ColumnConfigs="@miConfiguracion">
</EntityTable>
```

### 2. Configurar Columnas

```csharp
var columnConfigs = new List<ColumnConfig<MiEntidad>>
{
    // CheckBox filter para booleans
    new()
    {
        Property = nameof(MiEntidad.Active),
        Title = "Estado",
        UseCheckBoxFilter = true,
        CheckboxFilterOptions = new CheckboxFilterConfig
        {
            MaxItems = 2,
            EnableSearch = false
        }
    },
    
    // CheckBox filter para strings con valores limitados
    new()
    {
        Property = nameof(MiEntidad.Category),
        Title = "Categoría", 
        UseCheckBoxFilter = true,
        CheckboxFilterOptions = new CheckboxFilterConfig
        {
            MaxItems = 50,
            EnableSearch = true,
            CacheTimeout = TimeSpan.FromMinutes(5)
        }
    },
    
    // Traditional filter para texto libre
    new()
    {
        Property = nameof(MiEntidad.Description),
        Title = "Descripción",
        UseCheckBoxFilter = false // ← Default behavior
    }
};
```

## 📊 Tipos de Filtros

### CheckBox Filters 📋

**Ideales para:**
- ✅ Campos booleanos (Active/Inactive)  
- ✅ Enums (Status, Type, Category)
- ✅ Strings con valores limitados (Country, Department)
- ✅ Propiedades relacionadas (Category.Name, User.Role)

**Características:**
- Lista desplegable con checkboxes
- Búsqueda interna opcional
- Soporte para "Select All" / "Clear All"
- Virtualización para listas grandes
- Cache inteligente

### Traditional Filters 🔍

**Ideales para:**
- ✅ Texto libre (Description, Comments)
- ✅ Números (Price, Quantity, ID)
- ✅ Fechas (CreatedDate, ModifiedDate)
- ✅ Campos únicos o con muchos valores

**Características:**
- Input tradicional
- Operadores (=, !=, >, <, contains, etc.)
- Filtros de rango para fechas/números

## 🔧 Configuración Avanzada

### CheckboxFilterConfig

```csharp
public class CheckboxFilterConfig
{
    // Tiempo de cache para valores únicos
    public TimeSpan CacheTimeout { get; set; } = TimeSpan.FromMinutes(5);
    
    // Máximo número de items a mostrar
    public int MaxItems { get; set; } = 100;
    
    // Habilita búsqueda dentro del filtro
    public bool EnableSearch { get; set; } = true;
    
    // Carga valores solo cuando se abre el filtro
    public bool LoadOnDemand { get; set; } = true;
    
    // Habilita virtualización para listas grandes  
    public bool EnableVirtualization { get; set; } = true;
    
    // Incluye valores null/empty
    public bool IncludeNullValues { get; set; } = true;
    
    // Texto para valores null
    public string NullValueText { get; set; } = "(Vacío)";
}
```

### Parámetros de EntityTable

```xml
<EntityTable T="MiEntidad"
             
             @* Habilitar sistema híbrido *@
             EnableHybridFilters="true"
             DefaultFilterMode="FilterMode.CheckBoxList"
             
             @* Configuración de cache *@
             EnableFilterCache="true"
             FilterCacheTimeout="TimeSpan.FromMinutes(5)"
             
             @* Handler personalizado (opcional) *@
             OnLoadColumnFilterData="@MyCustomHandler">
</EntityTable>
```

## ⚡ Performance y Cache

### Cache Automático

El sistema incluye cache inteligente para mejorar performance:

```csharp
// Cache se maneja automáticamente
// - Se cachea por propiedad + filtro de búsqueda
// - Expira según FilterCacheTimeout
// - Se limpia automáticamente cuando se excede límite

// Métodos públicos disponibles:
await entityTable.ClearFilterCache(); // Limpiar todo
await entityTable.ClearFilterCache("PropertyName"); // Limpiar específico
var stats = await entityTable.GetFilterCacheStats(); // Estadísticas
```

### Optimizaciones Automáticas

- **Lazy Loading**: Datos se cargan solo al abrir filtros
- **Debouncing**: Búsquedas se ejecutan con delay para evitar spam
- **Virtualización**: Listas grandes se virtualizan automáticamente
- **Request Batching**: Múltiples filtros se pueden batchar

## 🎨 Personalización

### Handler Personalizado

```csharp
private async Task HandleCustomFilterData(DataGridLoadColumnFilterDataEventArgs<MiEntidad> args)
{
    var propertyName = args.Column?.GetFilterProperty();
    
    switch (propertyName)
    {
        case nameof(MiEntidad.Status):
            // Personalizar valores de enum con iconos
            args.Data = new[]
            {
                new { Value = Status.Active, Text = "✅ Activo" },
                new { Value = Status.Inactive, Text = "❌ Inactivo" },
                new { Value = Status.Pending, Text = "⏳ Pendiente" }
            };
            args.Count = 3;
            break;
            
        case nameof(MiEntidad.Priority):
            // Ordenar por importancia personalizada
            var priorities = await GetPrioritiesFromDatabase();
            args.Data = priorities.OrderBy(p => p.ImportanceLevel)
                                 .Select(p => new { Value = p.Id, Text = p.Name });
            args.Count = priorities.Count();
            break;
            
        default:
            // Usar auto-generación para otros campos
            return;
    }
}
```

### Personalizar Texto de Valores

```csharp
// Personalizar cómo se muestran los valores booleanos
```

```csharp
// ProductList.razor.cs
public partial class ProductList
{
    private EntityTable<Product>? productTable;
    
    private async Task HandleFilterData(DataGridLoadColumnFilterDataEventArgs<Product> args)
    {
        var property = args.Column?.GetFilterProperty();
        
        // Personalizar solo campos específicos
        if (property == nameof(Product.IsActive))
        {
            args.Data = new[]
            {
                new { Value = true, Text = "✅ Disponible" },
                new { Value = false, Text = "❌ Descontinuado" }
            };
            args.Count = 2;
        }
        // Para otros campos, usar auto-generación
    }
    
    private async Task ClearFiltersCache()
    {
        await productTable!.ClearFilterCache();
    }
}
```
